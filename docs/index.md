# Soundboard MAUI

.NET SDK and reference client for streaming voice engines.

## Key Features

- **Standalone SDK** — `Soundboard.Client` works in console, WPF, ASP.NET, or any .NET 8+ project
- **Zero UI Dependencies** — Pure C# library for engine communication
- **Streaming Speech** — WebSocket-based voice streaming with health checks
- **NAudio Adapter** — PCM16 playback for Windows
- **Reference Apps** — MAUI desktop and console CLI included

## NuGet Packages

```bash
dotnet add package Soundboard.Client
dotnet add package Soundboard.Maui.Audio
```
